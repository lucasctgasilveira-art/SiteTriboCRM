Declaração de Propósito Único

📅 Versão 1.0 · Abril de 2026 · Extensão TriboCRM para WhatsApp Web

Este documento atende à exigência da Chrome Web Store de declaração clara e única do propósito da Extensão, conforme as políticas do Programa de Desenvolvedores do Google Chrome.

Propósito único

A Extensão TriboCRM para WhatsApp Web tem o propósito único de permitir que usuários autenticados do TriboCRM visualizem e gerenciem os dados do seu próprio CRM diretamente sobre a página do WhatsApp Web, eliminando a necessidade de alternar entre abas do navegador durante atendimentos comerciais.

Como isso se traduz em funcionalidades

Para cumprir esse propósito único, a Extensão:

  • Adiciona um painel lateral à página web.whatsapp.com
  • Identifica o contato da conversa atualmente aberta (nome exibido e, quando disponível, número de telefone)
  • Busca esse contato na conta do TriboCRM do Usuário através de uma API autenticada
  • Exibe os dados do lead correspondente no painel lateral
  • Permite o registro manual de interações e o uso de modelos de mensagem pré-configurados

Justificativa das permissões solicitadas

A Extensão solicita apenas as permissões estritamente necessárias para cumprir seu propósito único:

storage — armazenar localmente o token de autenticação, preferência de tema e estado do painel. Necessário para manter o Usuário logado entre sessões e preservar suas preferências visuais.

alarms — executar verificações periódicas de mensagens agendadas pelo Usuário. Necessário para a funcionalidade de envio programado, que permite ao Usuário planejar follow-ups.

notifications — exibir notificações do sistema quando mensagens agendadas são processadas. Auxilia o Usuário a acompanhar ações em segundo plano da Extensão.

host_permissions: https://web.whatsapp.com/* — injetar o painel lateral na página do WhatsApp Web. Essa é a razão de ser da Extensão.

host_permissions: https://api.tribocrm.com.br/* — comunicar-se com o backend do TriboCRM para autenticação e operações de leads. Necessário para buscar e atualizar dados do CRM do Usuário.

Uso de código remoto

A Extensão não executa código remoto. Todo o código JavaScript é empacotado no momento da publicação e distribuído via Chrome Web Store. Nenhum script externo é carregado em tempo de execução.

Uso de dados do usuário

Detalhes completos sobre dados coletados, armazenados e transmitidos estão na Política de Privacidade da Extensão.

Em resumo:

  • A Extensão lê apenas o contato da conversa ativa no WhatsApp Web
  • A Extensão não lê, armazena ou transmite o conteúdo de mensagens
  • A Extensão não envia mensagens automaticamente
  • A Extensão não captura áudio, vídeo, imagens ou outros arquivos de mídia
  • A Extensão não coleta a lista de contatos do Usuário
  • A Extensão não vende dados a terceiros

Consentimento do usuário

Na primeira abertura, a Extensão apresenta ao Usuário uma tela de aceite com dois checkboxes obrigatórios:

  1. Aceite dos Termos de Uso da Extensão
  2. Consentimento com a Política de Privacidade da Extensão (conforme LGPD)

A Extensão não funciona até que ambos os aceites sejam dados expressamente. O aceite é registrado localmente com a versão dos termos vigente. Alterações relevantes nos termos disparam novo aceite.
